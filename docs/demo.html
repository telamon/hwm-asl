Why?

Make PoW slightly sustainable
This is a Schnorr key generator based on a spec that encodes ASL into the first 19-bits of your public key.
It uses the same mechanics as Proof of Work but burns data instead of burning zeros.
True Decentralized Taxonomy
By redistributing the data across popular metrics.
We can holistically sort the hashspace, letting clients attract other clients with similar interests.
Forming natural clusters.
Localize relays
The big global relays are not bad yet, we need them. But we also need many more local actors to translate and import culture.
So In order to relive the burden of event parsing and sustain network growth: let relays that appeal more to some audiences and less to others emerge.

How to

Install
$ npm i hwm-asl latlon-geohash geolib
What you get
hwm-asl
Provides decodeASL() and roll() functions
latlon-geohash
Transcoder between Latitude + Longitude pair and Geohash
geolib
Contains helpers to calculate distance using metric the system
Decode

Pass any 32byte public key or 64char hexstring to the decode-function

import { decodeASL } from 'hwm-asl'
import Geohash from 'latlon-geohash'

const npub = '0149170fe78b061ce6c7295fff2daa303f710ba17efd8fafd8343292b4295e84'

const { age, sex, location } = decodeASL(npub)

console.log('Age:', [16, 24, 32, 42][age])
console.log('Sex:', ['Sherry', 'Herman', 'Norton', '0dessa'][sex])
console.log('Location:', Geohash.decode(location, 3))
Produces: 
Age: 24
Sex: Shreman
Location: { lat: -54.1, lon: -130.1 }
